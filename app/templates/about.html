About the project

Built for the modern web

Notes Studio is a full-stack notes app built to demonstrate production-grade development — from secure auth to containerized deployment.

⚡

FastAPI

High-performance Python backend with automatic OpenAPI docs and async support.

🐘

PostgreSQL

Reliable relational database with SQLAlchemy ORM for clean data modeling.

🔐

JWT Auth

Secure token-based authentication with bcrypt password hashing.

🐳

Docker

Multi-stage builds for minimal images. Full stack runs with one command.

🔁

Nginx

Reverse proxy with load balancing across two API instances for reliability.

🚀

CI/CD

GitHub Actions pipeline — test, build Docker image, and deploy automatically.
